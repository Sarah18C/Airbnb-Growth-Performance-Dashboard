# Airbnb Performance Dashboard🏠📊

An interactive Power BI dashboard designed to analyze Airbnb listings, booking trends, pricing patterns, and host performance. This dashboard helps property owners, investors, and analysts understand the short-term rental market and make data-driven decisions to maximize revenue and occupancy.



## 🚀 Project Objective


The goal of this project is to:

  * Analyze Airbnb listing distribution across different locations
  
  * Monitor pricing trends and revenue performance
  
  * Evaluate occupancy rates and booking patterns
  
  * Understand the impact of property type and location on demand
  
  * Support data-driven decisions for hosts and property investors

## 🧮 Key Performance Indicators (KPIs)

  1] Total Listings
  
  2] Total Revenue
  
  3] Average Daily Rate (ADR)
  
  4] Average Occupancy Rate
  
  5] Average Review Score
  
  6]Total Bookings

These metrics help evaluate property performance and market demand.


## 📊 Dashboard Features
# 1. Overview Dashboard

Provides a high-level summary of Airbnb performance including:

  1) Total number of listings
  
  2) Total revenue generated
  
  3) Average daily price trends
  
  4) Occupancy rate analysis
  
  5) Booking trends over time

# 2. Property & Pricing Analysis

Provides insights into pricing and property performance including:

  1) Average price by property type
  
  2) Revenue contribution by room type
  
  3) Pricing distribution across locations
  
  4) Comparison of different property categories

# 3. Host & Customer Insights

Provides deeper insights into host activity and customer satisfaction including:

  1) Host response time impact on reviews
  
  2) Review score analysis
  
  3) Booking trends by property type
  
  4) Customer review distribution
  

## 🎯 Business Use Case

This dashboard helps Airbnb hosts, analysts, and investors to:

  * Identify profitable locations for short-term rentals
    
  * Optimize pricing strategies for higher revenue
    
  * Understand booking demand and occupancy trends
    
  * Improve customer satisfaction through review analysis
    
  * Make better investment decisions in the rental market
    

## 🛠 Tools & Technologies Used

  Microsoft Power BI – Dashboard Development & Visualization
  
  Power Query – Data Cleaning & Transformation
  
  DAX – Measures and Calculations
  
  Excel / CSV – Dataset
  
  Data Modeling – Relationships between tables

## 📈 Key Insights Generated

1) Certain neighborhoods generate higher revenue and bookings
2) Entire homes generally have higher prices compared to private rooms
3) Higher review scores often correlate with better booking rates
4) Pricing trends vary depending on location and property type
5) Seasonal trends influence Airbnb booking demand
